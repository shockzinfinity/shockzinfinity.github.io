## selenium capabilities 관련 오류 발생 시

- Edge on Mac
```python
from selenium import webdriver

edge_options = {
    "executable_path": "/Users/temp/selenium/edgedriver_mac64/msedgedriver",
    "capabilities": {
        "platformName": 'mac os x'
    }
}

browser = webdriver.Edge(**edge_options)
```
